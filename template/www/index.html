zkWasm SDK

import { Module, verify } from 'https://unpkg.com/zkwasm@0.1.6?module';

const res = await fetch('/pkg/release.wasm');
const binary = await res.arrayBuffer();
let m = await Module.fromBinary(binary);

let result = await m.invokeExport('greet', [new TextEncoder().encode('John')]);

const verify = await verify(result.proof);

How it works

zkWasm is a zero-knowledge proof system for WebAssembly. It allows you to prove that a WebAssembly program was executed correctly without revealing any information about the program.

Start by editing your proof logic in the wasm/assembly/index.ts AssemblyScript file and you can change how it's called in the www/index.mjs file.
